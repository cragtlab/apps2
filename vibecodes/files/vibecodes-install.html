Step 2 — Click it on any page & open your workspace
Click Open Directory and pick your workspace folder.
tasks.csv, archive.csv and the files/ subfolder are created automatically.
Drop your HTML/JS files into files/, add tasks, and the workflow begins.
Step 3 — The manual LLM loop
① Task picked up → full prompt shown in a modal
② Hit ⇸ Copy All → paste into your LLM chat
③ Click I’ve sent it → paste modal opens
④ Paste the LLM’s raw HTML → 💾 Save to File
⑤ File written to files/ → task marked Done
Workspace layout
your-workspace/
  tasks.csv — active task queue
  archive.csv — archived tasks
  files/ — your HTML & JS files (read + written directly)
Requires a Chromium browser with File System Access API support. Toggle panel with the bookmarklet.
